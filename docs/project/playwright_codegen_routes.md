# Playwright Codegen Routes

Playwright `codegen` here нужен не для постоянных автотестов, а для быстрой записи пользовательского маршрута кликов в игре.

Типовой маршрут:
- Главное меню
- Новая игра
- Стандартная карта
- Зелёная фракция
- Игровая карта

## Как запустить

1. Поднимите локальный сервер игры из корня проекта:

```powershell
py -m http.server 8000
```

2. Запустите codegen:

```powershell
npx.cmd playwright codegen http://127.0.0.1:8000/index.html
```

3. После открытия браузера:
- вручную пройдите нужный маршрут;
- копируйте сгенерированный код из окна Playwright;
- сохраните его в `tests/<route_name>.spec.js`;
- затем при необходимости упростите координаты, ожидания и лишние шаги.

## Когда использовать

- настройка направления юнитов;
- проверка посадки и grounding;
- проверка UI-маршрутов;
- воспроизведение багов кликами;
- визуальная проверка пыли и теней.

## Когда не использовать

- простые числовые правки;
- проверка точных PNG-файлов;
- задачи, где достаточно `node --check`.

## Удобный запуск

Для быстрого старта есть launcher:

```text
07_PLAYWRIGHT_CODEGEN.bat
```

Он не меняет существующие тесты и не создаёт visual regression snapshots сам по себе. Это только удобная точка входа для записи маршрута.

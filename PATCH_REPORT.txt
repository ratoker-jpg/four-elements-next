- rawDir=3 (SW) → sprite=6 (смотрит NE) — юнит едет боком/назад
- rawDir=7 (NE) → sprite=2 (смотрит S)  — юнит едет боком

Что исправлено:

1. FE_SCOUT_DIR_MAP (runtime_flags.js):
   - Старый: [2, 0, 0, 6, 6, 4, 4, 2] (копия harvester)
   - Новый:  [7, 0, 1, 2, 3, 4, 5, 6]
   - Формула: DIR_MAP[rawDir] = (rawDir + 7) % 8
   - Компенсирует поворот спрайтов на +1 CW

2. unitDir8 forceDir (main.js):
   - Добавлен scout и light_tank в цепочку forceDir внутри unitDir8()
   - Раньше forceDir применялся только к harvester и builder
   - Теперь FE_SCOUT_FORCE_DIR и FE_LIGHT_TANK_FORCE_DIR работают корректно
     при dev-калибровке через горячие клавиши

- rawDir=1 (SE) → sprite=0 → faces (0+1)%8=1=SE ✓
- rawDir=2 (S)  → sprite=1 → faces (1+1)%8=2=S  ✓
- rawDir=3 (SW) → sprite=2 → faces (2+1)%8=3=SW ✓
- rawDir=4 (W)  → sprite=3 → faces (3+1)%8=4=W  ✓
- rawDir=5 (NW) → sprite=4 → faces (4+1)%8=5=NW ✓
- rawDir=6 (N)  → sprite=5 → faces (5+1)%8=6=N  ✓
